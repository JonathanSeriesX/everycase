# iPhone 16e & 17e case lineup

### Compatibility

The iPhone 16e Silicone Case can be used with iPhone 17e, but it lacks MagSafe magnets.

Non-Beats iPhone 17e cases are compatible with [Crossbody Straps](/others/crossbody).

iPhone 17e supports MagSafe, so it happily takes a [MagSafe Wallet](/others/wallet).
